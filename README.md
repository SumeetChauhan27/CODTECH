## 📂 Project Portfolio

### 1️⃣ [Data Visualization Dashboard](./codtech-data-viz-dashboard)
A highly interactive, fully responsive analytical interface built with React, Vite, Recharts, and Nivo.
* **Key Features:** Dynamic light/dark themes, customizable accent colors, global user preferences (via `localStorage`), pagination, search/filtering of datasets, and robust CSV export functionality.
* **Tech Stack:** React 19, Tailwind CSS v4, Recharts, Nivo, Lucide Icons.
* **Focus:** Complex state management, performance optimization for data rendering, and advanced UI/UX theming.

### 2️⃣ [Full-Stack MERN Blog](./codtech-mern-blog)
A complete full-stack blogging platform featuring secure authentication and robust content management.
* **Key Features:** Secure JWT-based user authentication, password hashing with bcrypt, protected routes, and complete CRUD (Create, Read, Update, Delete) capabilities allowing users to publish and manage their own articles.
* **Tech Stack:** MongoDB Atlas, Express.js, React (Vite), Node.js, Tailwind CSS.
* **Focus:** Building scalable RESTful APIs, secure user authorization, and connecting a frontend to a cloud NoSQL database.

### 3️⃣ [Real-Time Chat Application](./codtech-realtime-chat-app)
A premium, real-time messaging workspace featuring instant synchronization and advanced chat features.
* **Key Features:** Google & Email authentication via Firebase, private room creation with 10-digit invite codes, WhatsApp-style read receipts (sent/delivered/read), emoji reactions, message pinning, and real-time online/away presence tracking.
* **Tech Stack:** React 19, Firebase Auth, Firebase Firestore, Tailwind CSS, Framer Motion.
* **Focus:** WebSockets/real-time listeners (`onSnapshot`), complex data synchronization, and highly interactive micro-animations.

### 4️⃣ [Next.js E-Commerce Platform](./codtech-nextjs-ecommerce)
A high-performance online storefront built leveraging modern Next.js App Router capabilities.
* **Key Features:** Global Shopping Cart and Wishlist Contexts, real-time dynamic product filtering with URL state syncing, beautiful UI layouts, and a multi-step simulated checkout and payment flow.
* **Tech Stack:** Next.js 15 (App Router), React, Tailwind CSS.
* **Focus:** Server Components vs Client Components, routing mechanics, SEO optimization, and building accessible, conversion-optimized user interfaces.
